# Электронная библиотека

Flask-приложение для автоматизированной информационной системы «Электронная библиотека».

## Тестовые пользователи

Для проверки работы проекта можно использовать следующие учётные записи:

| Роль          | Логин   | Пароль        |
| ------------- | ------- | ------------- |
| Администратор | `admin` | `password123` |
| Модератор     | `moder` | `password123` |
| Пользователь  | `user1` | `password123` |
